Instance Method Details

# (void) show(translation = 'new')

Show editor window. If language cod is set to 'new' (default) Editor will be in 'new language' mode. Otherwise, editor will open language to translate by its code.

Parameters:

  • translation ( String ) — language code of translation to edit. Default is 'new'.

# (void) _setEditorFor(lang, initialValue = '') Private

Set appropriate editor to display and edit translation

Parameters:

  • lang ( String ) — Language code
  • initialValue ( String ) — initial value to show in editor

# (void) _createLangList() Private

Note: List of all available languages is set by initialization method

Create drop-down box with list of all available languages

# (void) removeLang(langCode)

Completely removes language translation. Language is still avaiable for further translate.

Parameters:

  • langCode ( String ) — Code of language to remove from list of translated languages

# (void) addLang(langCode, translation)

Note: Usefull for editing translations using external extensions

Programatically add translation for existing langCode

Parameters:

  • langCode ( String ) — Language code in ISO format
  • translation ( String ) — Translated string
